# 🌶️ Spice Route: Restaurant Management System

A premium, full-stack restaurant menu and order management system built with Flask. This application streamlines the dining experience from customer selection to kitchen fulfillment.

### 📝 Short About
**Spice Route** is a modern web application designed for boutique Indian restaurants. It provides a dual-interface system: a vibrant, interactive menu for customers to place orders and a real-time dashboard for chefs to manage the kitchen flow. The system features automatic tax calculations, instant digital receipts, and a professional print-ready billing system.

---

### ✨ Key Features

- **Interactive Customer Menu**: A responsive, visually stunning menu with real-time cart management.
- **Instant Digital Billing**: Automatically generates a professional receipt upon order placement.
- **One-Click Printing**: Optimized print styles for thermal receipt printers or standard A4.
- **Kitchen Dashboard**: Live order tracking for chefs with status updates (Pending, Cooking, Ready).
- **GST Calculation**: Automated 5% GST calculation for accurate financial summaries.
- **Premium UI/UX**: Built with a modern aesthetic using Inter typography and a vibrant color palette.
- **Vercel Ready**: Fully configured for seamless cloud deployment.

### 🚀 Getting Started

#### Prerequisites
- Python 3.8+
- Pip (Python package manager)

#### Local Installation
1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd RESTAURANT-MENU
   ```

2. **Set up a Virtual Environment (Recommended):**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application:**
   ```bash
   python indian_bistro/app.py
   ```
   The app will be available at: `http://127.0.0.1:5001`
